In questa pagina viene usato il css internal

Con il CSS internal, lo stile è presente nella stessa pagina html (come per l'inline).
A differenza dell'inline, tutto il CSS viene raggruppato sotto il tag style all'interno dell'head.

Anche in questo caso quindi, non separiamo completamente il contenuto dallo stile, ma la pagina inizierà ad essere molto più ordinata.

Dobbiamo indicare:

  • il selettore (tag)
  • apertura perentesi graffa
  • le coppie chiave-valore separate da punto e virgola delle varie proprietà css che vogliamo usare
  • chiusura parentesi graffa
