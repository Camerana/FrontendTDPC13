inline

In questa pagina viene usato il css inline (come abbiamo fatto fin'ora).

Significa impostare lo stile nella stessa pagina html.

Non è il metodo ottimale in quanto così facendo, non separiamo il contenuto dallo stile.

Il CSS inline è usato attraverso l'attributo style all'interno dei vari tag.

Guardando il codice della pagina non riusciamo a distinguere ed individuare ad una prima occhiata il contenuto e lo stile...

Apri il sito con notepad++ o Visual Studio Code per vedere il codice delle pagine.
