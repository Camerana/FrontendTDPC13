external

In questa pagina viene usato il css external.

E'il metodo ottimale in quanto separiamo il contenuto dallo stile.

Questo metodo prevede di avere un file esterno in cui inseriamo tutti i nostri stili.

Ad esempio creiamo una cartella style e al suo interno il file myStyle.css

Nella pagina html, richiamiamo il file esterno con l'uso del tag link (nel tag head)

Apri il sito con notepad++ o Visual Studio Code per vedere il codice delle pagine.
