IMMAGINI:
	<img> --> non vuole la chiusura
ATTRIBUTI:
	src="" --> url assoluto (https://...) o percorso relativo (images/imm.jpg)
				- il path relativo si basa sul livello in cui è presente il file html attuale:
				  esempio 1: file html e immagine nella stessa cartella: src="carbonara.jpg"
				  esempio 2: file html nella root e immagine in una sottocartella images: src="images/carbonara.jpg"				  
	
	alt="" --> testo alternativo --> visualizzato solo se l'immagine non è visualizzata.
	
	Se non specifichiamo le dimensioni, l'immagine è visualizzata nella sua dimensione naturale
	Se vogliamo ridimensionarla:
	width="" 
	height=""
	(se si usano entrambi, si perde il rapporto fra le 2 dimensioni)
	
	esempio: se scarico un immagine 500px di larghezza e 100px di altezza,
			 e imposto width ed height a 200px, l'immagine risulterà quadrata
			 e "stirata" maggiormente in altezza.

***
TAG INLINE VS TAG BLOCK:

	- BLOCK --> esempio:
		<p>paragrafo uno</p>
		<p>paragrafo due</p>		
		
		I due paragrafi sono visualizzati uno sotto l'altro:
		Alcuni tag (come p), detti BLOCK, lasciano sopra e sotto di essi una linea.
	- INLINE --> esempio:
		<img src="carbonara.jpg">
		<img src="spritz.jpg">
		
		Le due immagini, a differenza dei p, sono visualizzate una accanto all'altra!
		Altri tag, detti INLINE, non lasciano spazi né sopra né sotto di essi.
