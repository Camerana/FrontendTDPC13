Come integrare Javascript in una pagina web

Inline (sconsigliato, da usare in rari casi)

Aggiungendo ad un elemento html (ad esempio un p) un attributo come onclick=""
e all'interno delle parentesi inserire il codice JavaScript, ad esempio:
alert('ciao, sono un alert');

Internal

Come ultimo elemento prima del tag di chiusura del body, inserire il tag script senza attributi, ma con contenuto del codice JavaScript come ad esempio il solito: alert('ciao, sono un alert');

External (consigliato)

Come facciamo per il css, anche per gli script, creare una sottocartella script nella quale inserire un file, ad esempio myScript.js
Come ultimo elemento prima del tag di chiusura del body, inserire il tag script senza contenuto, ma con l'attributo src="" contenente il percorso che porta al nostro script esterno.
